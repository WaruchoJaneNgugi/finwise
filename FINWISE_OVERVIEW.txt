----------------------------------------------------------------
WHAT IS FINWISE?
----------------------------------------------------------------
FinWise is a personal finance web application built specifically
for Kenyans. It helps users track expenses, manage bills, grow
investments, set savings goals, and get AI-powered financial
advice — all in one place. Data is PIN-protected and works
offline. No bank login is required.

Tagline: "Your Money, Mastered"

----------------------------------------------------------------
CORE FEATURES (available across tiers)
----------------------------------------------------------------
- Expense tracking & categorisation (housing, food, transport,
  utilities, medical, education, entertainment, shopping, etc.)
- Financial Advisor view with personalised tips
- Dashboard overview with monthly spending summary
- Bills & recurring payments tracker
- Savings goals with deadlines and progress tracking
- Emergency fund tracker
- Net Worth calculator (assets vs liabilities)
- Investment portfolio manager (SACCO, MMF, stocks, bonds,
  crypto, pension, fixed deposits, savings accounts)
- Spending insights & analytics
- AI Chat financial advisor
- CSV data exports
- Alerts & SOS emergency system with contact notifications
- Spending habits tracker
- Financial alerts panel

ADMIN PANEL
----------------------------------------------------------------
A separate admin interface (not visible to regular users) allows
the FinWise team to manage the platform.

Admin Roles & Access:
-------------------------------------------------
| Section        | super_admin | support | finance |
|----------------|-------------|---------|---------|
| Overview       |     ✓       |    ✓    |    ✓    |
| Users          |     ✓       |    ✓    |    ✗    |
| Subscriptions  |     ✓       |    ✗    |    ✓    |
| Settings       |     ✓       |    ✗    |    ✗    |
-------------------------------------------------

Role Descriptions:

  SUPER ADMIN
  - Full access to all admin sections
  - Can create and delete other admin accounts
  - Can change any user's subscription tier
  - Can blacklist or delete user accounts
  - Manages admin roles (support, finance, super_admin)

  SUPPORT
  - Views platform overview (user counts, tier distribution)
  - Manages users: search by name/phone, view user details,
    blacklist users (read + limited write)
  - Cannot access subscriptions or admin settings

  FINANCE
  - Views platform overview
  - Views subscription data: per-tier user counts,
    Monthly Recurring Revenue (MRR) estimates
  - Cannot manage users or admin settings

Admin Panel Sections:

  Overview
  - Total user count
  - Breakdown of users per tier (Free, Silver, Gold, Platinum)
  - Visual tier distribution bar chart

  Users
  - Full user list with search (name or phone)
  - User detail drawer: view profile, financial snapshot
  - Super admin can change tier, blacklist, or delete a user
  - Support role can view but not edit tiers

  Subscriptions
  - Per-tier user counts and individual MRR contribution
  - Total estimated Monthly Recurring Revenue (MRR) in KES

  Settings (super_admin only)
  - Create new admin accounts (email, password, role)
  - View all existing admin accounts
  - Remove admin accounts
